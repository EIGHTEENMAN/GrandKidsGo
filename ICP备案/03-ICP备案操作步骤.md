# 童慧行 ICP 备案操作步骤

**入口：** https://beian.aliyun.com
**前提：** 阿里云账号 + 已实名认证

---

## 第一步：登录阿里云备案系统

1. 打开 https://beian.aliyun.com
2. 用阿里云账号登录
3. 点击 **开始备案**

## 第二步：填写主办者信息

### 2.1 选择主办者性质

- **企业**（推荐）：营业执照主体
- **个人**：身份证主体

### 2.2 填写企业信息

- 企业名称（与营业执照完全一致）
- 营业执照注册号
- 法人姓名、身份证号
- 注册资本
- 营业期限
- 经营范围
- 通信地址（详细到门牌号）

### 2.3 上传证件

- 营业执照副本（彩色扫描，PDF 或 JPG）
- 法人身份证正反面（彩色扫描）

## 第三步：填写网站信息

### 3.1 网站基本信息

- **网站名称**：`童慧行`
- **网站服务内容**：
  - 勾选 **文化** 和 **教育**
- **网站语言**：中文
- **网站简介**：
  > 儿童学习平台。涵盖古诗词、国学经典、通识百科、英语单词等学习内容。所有内容免费，不含广告，不含 UGC。
- **服务类型**：**非经营性**

### 3.2 域名信息

- 输入 `grandand.com` 和 `www.grandand.com`
- 系统自动检测域名是否已实名
- **如未实名**：先在阿里云域名控制台做实名认证

### 3.3 接入信息

- 接入商：阿里云（自动）
- 接入方式：阿里云 ECS
- 服务器 IP：47.114.77.124
- 服务器放置地：浙江杭州

## 第四步：填写网站负责人

- 如网站负责人 = 法人：勾选"同主办者"
- 否则填写：
  - 姓名
  - 身份证号
  - 手机号（接收验证码）
  - 邮箱
  - 与主办者关系
  - **上传授权书**（员工需上传）

## 第五步：上传材料

按系统提示上传：

- [ ] 营业执照副本
- [ ] 法人身份证正反面
- [ ] 网站负责人身份证正反面（如非法人）
- [ ] 授权书（如网站负责人非法人）
- [ ] 域名证书（阿里云域名控制台下载）
- [ ] 核验单（系统生成，下载签字盖章后上传）

## 第六步：阿里云初审

- 提交后，阿里云在 **1-2 个工作日** 内初审
- 初审结果会通过短信、邮件、备案系统通知
- 退回修改：按提示修改后重新提交

## 第七步：幕布拍照（多数省份需要）

- 阿里云审核通过后，会**邮寄幕布**到网站负责人地址
- 收到幕布后：
  1. 网站负责人手持**身份证**
  2. 站在**幕布**前
  3. 拍摄**清晰照片**：人脸 + 身份证 + 幕布文字 都在画面内
  4. 上传到备案系统

**注：**部分省份（如浙江）已取消幕布拍照，改为**人脸核验**（App 在线完成）。

## 第八步：阿里云提交管局

- 阿里云将你的备案信息**提交到浙江省通信管理局**
- 提交时间：1-2 个工作日
- 提交后会给你一个**申请编号**（用于查询进度）

## 第九步：等待管局审核

- 浙江省通信管理局审核：**7-20 个工作日**
- 期间可能接到电话核实信息
- 审核结果：
  - **通过**：获得 ICP 备案号（浙ICP备 xxxxxxxx号）
  - **退回**：按提示修改后重新提交

## 第十步：备案成功

1. 收到备案成功通知
2. 备案号：`浙ICP备 + 数字 + 号`
3. 备案密码：用于后续变更备案

### 10.1 网站底部加备案号

```html
<a href="https://beian.miit.gov.cn/" target="_blank">
  浙ICP备 xxxxxxxx号-1
</a>
```

### 10.2 30 日内做公安联网备案

- 入口：https://beian.mps.gov.cn
- 用 ICP 备案号 + 备案密码登录
- 详细材料见《公安联网备案材料清单》

### 10.3 信息变更管理

- 备案信息变更 → 30 日内变更备案
- 域名变更 → 30 日内变更备案
- 服务器变更（迁移）→ 30 日内变更备案
- 主体变更（公司转让）→ 注销原备案，重新备案

## 常见问题

### Q1：个人可以备案吗？

A：可以，但：
- 备案后网站不能用于商业用途
- 部分省份需要居住证
- 企业备案更稳定（推荐）

### Q2：备案后多久必须开通网站？

A：无硬性要求，但建议 **30 日内** 开通，否则可能被管局注销。

### Q3：可以同时备案多个网站吗？

A：可以。一主体最多 20-50 个网站（看省份）。

### Q4：grandand.com 的子域名需要单独备案吗？

A：
- **同一站点下**的子域不需要单独备案（如 xueshici.grandand.com 作为"童慧行"站点的一个子模块）
- **作为独立站点**的子域需要单独备案（如 admin.grandand.com 作为独立内部系统）

### Q5：备案号 -1、-2 是什么意思？

A：每个独立网站对应一个编号：
- 主站 grandand.com → 浙ICP备 xxxxxxxx号**-1**
- 子站 admin.grandand.com → 浙ICP备 xxxxxxxx号**-2**
- 等等

### Q6：备案失败后多久能重新申请？

A：立即可重新申请（修改后）。

### Q7：备案信息错误怎么办？

A：在阿里云备案系统提交"变更备案"。

### Q8：网站负责人变更怎么办？

A：网站负责人变更后，**15 日内**提交变更备案。

### Q9：营业执照变更怎么办？

A：营业执照变更后，**30 日内**提交变更备案。

### Q10：服务器迁出阿里云怎么办？

A：迁出前在阿里云做"转出接入"，到新服务商做"转入接入"。
