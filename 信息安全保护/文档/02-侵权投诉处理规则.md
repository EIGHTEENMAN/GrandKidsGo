# 童慧行侵权投诉处理规则

**生效日期：** 2026 年 7 月 2 日
**版本：** v1.0

本规则依据《中华人民共和国网络安全法》《信息网络传播权保护条例》《互联网信息服务管理办法》等法律法规制定，保障权利人合法权益。

## 一、受理范围

本平台受理以下投诉：

1. **著作权侵权**：未经授权的复制、发行、传播他人作品
2. **商标权侵权**：未经授权使用他人注册商标
3. **肖像权侵权**：未经本人同意使用肖像
4. **名誉权/隐私权侵权**：侮辱、诽谤、泄露隐私
5. **其他民事侵权**

## 二、受理方式

### 2.1 站内提交（推荐）

- 登录账号 → "帮助" → "侵权投诉"
- 管理员后台：grandand.com/admin

### 2.2 邮件提交

- 邮箱：[complaint@grandand.com]
- 邮件标题请注明"侵权投诉+被投诉内容ID"

### 2.3 举报 API（开发者）

```http
POST /api/moderation/report
Authorization: Bearer <user-token>
Content-Type: application/json

{
  "targetType": "poem|comment|post|user|...",
  "targetId": "<内容ID>",
  "reason": "侵权类型 (copyright|trademark|defamation|privacy|...)",
  "contentSnippet": "侵权内容摘要(选填)"
}
```

## 三、投诉人需提供材料

1. **身份证明**：个人提供身份证正反面，企业提供营业执照
2. **权属证明**：
   - 著作权：作品登记证书、原稿、首次发表证据
   - 商标权：商标注册证
   - 肖像权：本人身份证明
3. **侵权事实**：
   - 被投诉内容标题、ID、URL
   - 侵权内容截图
   - 侵权比对说明
4. **真实性承诺**：声明所提供材料真实、合法

## 四、处理流程

| 阶段 | 时限 | 操作 |
|---|---|---|
| 1. 受理 | 1 个工作日 | 审核材料完整性 |
| 2. 初步审查 | 3 个工作日 | 判断是否构成侵权 |
| 3. 通知被投诉人 | 1 个工作日 | 给予陈述申辩机会 |
| 4. 调查核实 | 5 个工作日 | 调取证据、技术比对 |
| 5. 作出处理 | 3 个工作日 | 删除/屏蔽/驳回 |
| 6. 反馈 | 1 个工作日 | 告知投诉人处理结果 |

**总计不超过 15 个工作日**

## 五、处理措施

根据侵权严重程度，可采取以下措施：

1. **删除**：直接删除侵权内容
2. **屏蔽**：限制访问，搜索引擎不收录
3. **断开链接**：阻止特定来源的访问
4. **封禁账号**：永久或临时封禁违规账号
5. **保存证据**：依法向主管部门提供
6. **转交司法**：涉嫌犯罪的依法移送公安机关

## 六、申诉机制

被处理人对处理结果不服的，可通过以下方式申诉：

1. 提交申诉：登录账号 → "我的-违规记录" → 申诉
2. 申诉 API：`POST /api/moderation/appeal`
3. 申诉需提供：申诉理由、补充证据
4. 申诉受理时限：15 个工作日内重新审核
5. 申诉成功的：撤销处罚、恢复内容

## 七、未成年人特别保护

涉及 14 周岁以下未成年人的内容，**一律从严处理**：

1. 任何形式的对未成年人的侮辱、霸凌、骚扰 → 立即删除 + 永久封号
2. 涉及未成年人个人信息的内容 → 立即删除 + 报告监护人
3. 涉嫌针对未成年人的违法犯罪 → 立即报警

## 八、批量投诉

权利人可对批量侵权内容提交投诉清单，我们提供：

1. 优先处理通道
2. 7 个工作日内完成批量审核
3. 提供处理报告

## 九、滥用投诉的处理

恶意投诉、虚假投诉的处理：

1. 第一次：警告
2. 第二次：限制投诉权限
3. 第三次：永久封禁投诉功能
4. 涉嫌敲诈勒索的，依法追究法律责任
