## 当前产品形态分析

童慧行当前上线范围：**学习应用**（诗词 / 国学 / 通识 / 英语）+ **童慧行来挑战**（答题）。

不涉及以下业务：

- ❌ 商业广告
- ❌ 付费内容 / 虚拟商品
- ❌ 用户生成内容（UGC）
- ❌ 网络视听节目（诗配动画已下线）
- ❌ 时政新闻
- ❌ 网络游戏（挑战是答题，无版号问题）
- ❌ 直播 / 视频通话
- ❌ 电子商务
- ❌ 论坛 / 社区 / 笔记
- ❌ 移动 App（仅 Web）

**因此当前实际需要办的备案只有 2 项**：
- ICP 备案
- 公安联网备案

## 备案所需材料清单

### ICP 备案

- [ ] 主办单位证件（营业执照/身份证）
- [ ] 主办负责人证件
- [ ] 网站负责人证件
- [ ] 域名证书
- [ ] 接入服务商信息（阿里云）
- [ ] 网站备案信息真实性核验单

### 公安联网备案

- [ ] 主办单位证件

- [ ] 网站安全责任人身份证
- [ ] 应急联系电话
- [ ] 信息安全管理制度
