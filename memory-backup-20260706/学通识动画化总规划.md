# 学通识动画化总规划

> **目标**：把 242 个 topic 从"静态配图"升级为"可看可玩可理解"的多模态学习内容，
> 让孩子看 1 个动画 > 读 1 段文字。

**核心理念**：动画不是装饰，是教学工具——让看不见的（血液循环/水循环）看见、让抽象的（月相/杠杆）摸到。

---

## 一、分级标准（4 档）

| 档位 | 教学价值 | 实现成本 | 必做理由 |
|---|---|---|---|
| **S 镇站级** | 极高（看不懂就没法学） | 中-高 | 不做就是书，做了就是玩具 |
| **A 强烈推荐** | 高（动画=秒懂） | 低-中 | ROI 最高 |
| **B 锦上添花** | 中（图文也能讲） | 中 | 资源够再做 |
| **C 不必做** | 低 | — | 真实图/文字已够 |

## 三、A 级（28 个）—— 强烈推荐

### 自然科学（13）
- **物质三态** → 分子运动（固紧/液流/气飞）
- **声光电磁** → 声波/光波/电磁波横向波动
- **摩擦力** → 接触面颗粒放大，滑动 vs 滚动对比
- **浮力** → 物体入水时水被排开，浮力箭头
- **力与运动** → 自由落体+抛物线
- **进化与适应** → 物种形态渐变（鱼→两栖→爬行→哺乳）
- **食物链** → 能量在生态系统流动（草→兔→狐→狼）

### 人体（5）
- **骨骼系统** → 关节活动（膝/肘/肩）
- **肌肉系统** → 屈肌伸肌对抗收缩
- **大脑与神经** → 突触放电、神经信号传递
- **免疫系统** → 白细胞吞噬病毒
- **生长发育** → 细胞分裂动画

### 地球/宇宙（5）
- **宇宙探索** → 火箭分级、登月轨迹、阿波罗
- **时间与空间** → 四季形成（地轴倾斜+公转）
- **海洋世界** → 洋流/季风/水循环
- **极地世界** → 冰川移动、冰架断裂
- **自然灾害** → 台风/洪水/海啸动态演示

### 技术工程（5）
- **机器人** → 关节转动+传感器感知+决策
- **通信技术** → 电磁波从手机到基站
- **能源技术** → 水电/风电/光伏能量转换
- **古代发明** → 造纸/活字印刷/火药动画演示
- **计算机与互联网** → 数据包在网络中流动

## 四、B 级（50+）—— 锦上添花

- **历史人物** → AI 视频/关键帧动画（孔子周游、玄奘西行、郑和下西洋、岳飞抗金）
- **地理景点** → 360° 环视/缓慢推镜（长江、喜马拉雅、张家界）
- **艺术类** → 笔触动起来（书法挥毫、中国画点染、戏曲动作）
- **中国戏曲** → 脸谱变化、动作片段
- **民俗活动** → 舞龙舞狮、节气习俗
- **建筑艺术** → 长城/故宫建造过程
- **陶瓷艺术** → 拉坯、烧制流程

---

## 五、C 级（其余 100+）—— 不必做

- **安全教育**（消防/交通/防溺水/食品安全）→ 真实场景照片+文字警示更有效
- **礼仪规范**（家庭/学校/社交）→ 文字+真实图
- **急救常识** → 真人示范视频片段
- **微观概念**（原子/分子/基因）→ S/A 已覆盖

## 六、技术路线 3 选 1

| 路线 | 技术栈 | 单图成本 | 适用 |
|---|---|---|---|
| **A. CSS/SVG 动画** | `@keyframes` + SVG path + Lottie | **0 元** | 轨道/相位/位移/简单形变（70% 场景） |
| **B. Canvas/WebGL** | Canvas 2D / Three.js / PixiJS | **0 元** | 粒子/流体/3D（25% 场景） |
| **C. AI 视频** | MiniMax/可灵/Sora 视频生成 | **¥1-5/秒** | 复杂动态场景（5% 场景，镇站之宝） |

**默认全用 A+B 路线**（零额外成本），C 留给 3-5 个镇站视频。

## 七、代码架构（最小改动）

### 现状（已具备基础）
- **单点入口**：`src/components/KnowledgeIllustration.vue`（109 行）
- **媒体回退链**：`.jpg → .svg → 文字`（3 级）
- **引用方**：`App.vue` 两处（详情页 445、Reader 490），统一组件
- **零外部动画库**：`package.json` 只有 `vue@3.5 + pinyin-pro + vite`

### 改造方案（核心思路）

**新加 1 个 AnimationSlot 组件 + 1 个 animation 索引**：

```
src/components/
  KnowledgeIllustration.vue  ← 已存在，改 1 行：<img> → <AnimationSlot>
  AnimationSlot.vue          ← 新增，判断 topicId 是否在动画清单，是→渲染对应组件，否→原图回退
  animations/                ← 新增
    SolarSystem.vue          ← S 级 12 个
    EarthMoon.vue
    ...
    index.ts                 ← 动画注册表
```

**回退链升级为 4 级**：
```
组件动画（Canvas/SVG）→ .jpg 真图 → .svg 水墨 → 文字占位
```

**URL 规则保持不变**：
- topic 封面：`/images/knowledge/{topicId}.{jpg|svg}`
- 章节配图：`/images/sections/{parentTopicId}-{sectionId}.{jpg|svg}`

**数据驱动**：所有动画用数据 props 驱动（颜色/速度/大小），不做硬编码。

## 八、实施排期（建议 4 周）

| 阶段 | 时间 | 内容 | 交付 |
|---|---|---|---|
| **第 0 周** | 1 天 | 架构改造：`AnimationSlot` 组件 + 动画注册表 | 一个空架子，可零成本回退到原版本 |

| **第 2 周** | 5 天 | A 级 28 个（每天 5-6 个） | 28 个组件 |

**里程碑**：

- 第 4 周末：80+ 动画上线 → **平台差异化卖点**

## 九、性能与成本预估

| 项 | 估算 |
|---|---|
| 组件代码量 | 每个动画 50-200 行，80 个总计 ~10K 行 |
| 额外磁盘 | **0**（A/B 路线无图片） |
| 额外内存 | **0**（客户端运行） |
| 移动端兼容 | A 路线 99%（纯 CSS）；B 路线需测试 3D 性能 |
| API 成本（C 路线） | 3-5 个视频 × 30 秒 × ¥2/秒 = **¥180-300** |
| 总开发工作量 | 1 人 × 4 周 = **160 工时** |

## 十、风险与对策

| 风险 | 影响 | 对策 |
|---|---|---|
| 移动端卡顿（3D 行星） | 中 | 提供"静态降级"开关，低端机跳过动画 |
| 动画不准确（科学错误） | 高 | 邀请科学顾问/教师审核（家长群志愿者） |
| 与现有 SVG 资源冲突 | 低 | 不删 SVG，保留作 fallback |
| 组件多导致 bundle 大 | 中 | 按需懒加载 `import()` 动画组件 |
