## Store 架构

```
wordStore (核心 - 保留)
├── records: WordRecord[]          // 每个单词的掌握状态
├── recordAnswer(id, correct)      // 答题回调
├── getMasteredCount()             // 已掌握数
├── getAccuracy()                  // 正确率
└── SRS 间隔重复算法（升级）

studyStore (新建)
├── todayQueue: WordEntry[]        // 今日学习队列
├── currentIndex: number           // 当前单词位置
├── sessionStats: {                // 本次会话统计
│   correct, wrong, duration
│ }
├── todayStats: {                  // 今日累计
│   newWords, reviewed, sessions
│ }
├── streak: number                 // 连续天数
└── getDailyQueue(): Promise<...>  // 生成今日队列

settingsStore (新建)
├── dailyNewWords: number          // 每日新词数（默认 10）
├── dailyReviewWords: number       // 每日复习数（默认 5）
├── autoPlayAudio: boolean         // 自动播放
├── voice: string                  // 语音选择
└── save(), load()
```

## 数据流

```
┌─────────────────────────────────────────────────────┐
│  User Tap "Start Today" (StudyHome)                   │
└──────────────────────┬──────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────┐
│  studyStore.getDailyQueue()                          │
│    1. 从 wordStore 找 level=0 的 N 个新词             │
│    2. 从 wordStore 找到期复习的 M 个旧词（SRS 算法）  │
│    3. 随机打乱                                       │
│    4. 写入 studyStore.todayQueue                      │
└──────────────────────┬──────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────┐
│  DailyStudy 循环                                     │
│    for each word in todayQueue:                       │
│      StudyStep 4 步                                  │
│      wordStore.recordAnswer()                        │
│      记录到 studyStore.sessionStats                   │
└──────────────────────┬──────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────┐
│  StudyComplete                                       │
│    save sessionStats → localStorage                  │
│    更新 streak（如果今天还没学）                      │
│    检查成就解锁                                      │
└─────────────────────────────────────────────────────┘
```

## 视觉设计原则

- **大字号**：单词 64px+，例句 24px
- **高对比**：黑底白字 / 白底深字
- **大按钮**：最小 60×60px，儿童友好
- **少文字**：emoji 优先，文字辅助
- **进度可视**：环形进度条 + 数字
- **正向反馈**：每步成功后撒花 + 音效
- **无失败惩罚**：拼错不扣分，只提示
